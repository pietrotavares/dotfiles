# Shortcuts

## Navigation
`alt-↑`: scroll up
`alt-PgUp`: scroll up

`alt-↓`: scroll down
`alt-PgDn`: scroll down

## Clipboard
`alt-c`: copy to clipboard
`alt-v`: paste from clipboard
`alt-y`: copy url under cursor to clipboard
`alt-o`: copy output of some previous command (pops a little menu for you to select from) to clipboard

## Misc
`alt-shift-k`: increase font size
`alt-shift-j`: shrink font size
`alt-home`: return to default font size
